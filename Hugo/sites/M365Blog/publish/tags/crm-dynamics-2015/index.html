The zip generated by the export needs to be unzipped so that the path of the files can be passed to the method called to import the data.

CRM Dynamics 2015 "Error Uploading Report" caused by using Visual Studio 2015

When I was trying to upload a SSRS report to replace an existing report to CRM Dynamics 2015 solution, I got the error Error Uploading Report An error occurred while trying to add the report to Microsoft Dynamics CRM. The error message does not say much. I decided to use Visual Studio 2015 to edit the CRM report which has caused the issue. I compared the code behind of the previous report and new report and found that there are new XML elements in the report generated by visual studio 2015 (SQL Server Data Tools for Visual Studio 2015).
