Add or Update Image Column using Power Automate with image from the web

By Reshmee Auckloo

May 28, 2023

Add or Update Image Column using Power Automate

The blog post Update image in SharePoint/Microsoft Lists Image columns using Power Automate by Ganesh Sanap explains how to update an image column in SharePoint or Microsoft Lists using Power Automate. According to the blog post, the native “Create Item” and “Update Item” actions in Power Automate do not directly support adding or updating image columns. Instead, Ganesh suggests using the “Send an HTTP Request to SharePoint” action to achieve this.

This blog post covers how to download an image from a image url and add/update the image column.

Create a list with the following details

To try the steps use an existing list or create a brand new list with the following columns.

ColumnType
titleMultiple lines of text
mediaUrlMultiple lines of text
mediaimage

Populate both title and mediaUrl fields. In the example above I am using Instagram images url.

Ensure Assets library is created with the following PnP PowerShell

Connect-PnPOnline -Url https://contoso.sharepoint.com/sites/<siteName>/ -Interactive
$web= Get-PnPWeb
$web.Lists.EnsureSiteAssetsLibrary()
Invoke-PnPQuery

Power Automate flow

Go to Power Automate and create a new Instant cloud flow with Manually trigger a flow trigger.

  1. Add the action Get Items to get all items from the SharePoint List specifying the Site Address and List Name

  2. To upload the image to the Site Assets library, use the Create File action to save the image obtained from previous action. Set the Folder to SiteAssets/lists/ and the FileName to the Title of the list item suffixed with .png.

To get the listId navigate to the list settings and copy the list id from the URL.

Turn Allow Chunking off on the Create File action otherwise if the file already exists, the flow will fail.

  1. To save the image to column, use the Send an HTTP request to SharePoint action and amend the properties to the following

Method : POST Uri : _api/web/lists(’’)/items(@{items(‘Apply_to_each’)?[‘ID’]})/validateUpdateListItem() Headers: content-type : application/json;odata=verbose accept : application/json;odata=verbose

Body:

{"formValues":[{"FieldName":"media","FieldValue":"{\"type\":\"thumbnail\",\"fileName\":\"@{items('Apply_to_each')?['Title']}.png\",\"fieldName\":\"media\",\"serverUrl\":\"https://contoso.sharepoint.com\",\"serverRelativeUrl\":\"/sites/siteName@{outputs('Create_file')?['body/Path']}\"}","HasException":false,"ErrorMessage":null}],"bNewDocumentUpdate":false,"checkInComment":null}
