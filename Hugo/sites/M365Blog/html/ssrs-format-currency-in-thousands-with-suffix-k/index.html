SSRS Format Currency in thousands with suffix K

Currency values can be displayed in thousands with k suffix using the format ‘£’#,0,‘k’;(‘£’#,0,‘k’) Edit the Format property of the textbox with ‘£’#,0,‘k’;(‘£’#,0,‘k’) Running the report, all values will be displayed in thousands with K suffix and currency symbol as prefix.
