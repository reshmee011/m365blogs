I had a requirement to change group members security group which has contribute permissions at the site collection to have read permissions to one document library. The trick is to break inheritance on the document library and amend the role of the security group to Read. $web =Get-SPWeb -Identity $webUrl $docLibrary=$web.Lists.TryGetList($LibraryName) $docLibrary.BreakRoleInheritance($False) #get role definition from web: Full Control or Contribute or Read $roleDefinitions = "Read" #get site group $siteGroup = $web.
