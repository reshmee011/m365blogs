Create DevTest Labs in Azure

Azure DevTest Labs is available in UK South and UK West as from December 2016, in addition to the other 21 regions it has supported. The steps to create the DevTest lab are Login to Azure portal as administrator Click the green + New menu Type DevTest Labs into the search box Select DevTestLabs from the results page Click on Create from the Description page. The advantages using DevTest Labs as mentioned from the Description page are DevTest Labs helps developers and testers to quickly create virtual machines in Azure to deploy and test their applications.
