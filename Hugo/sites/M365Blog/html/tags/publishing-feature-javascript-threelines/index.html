Search Display Template js file not created when HTML file for Item_ThreeLines is created

On a SharePoint Online site collection where Publishing feature was initially turned on and subsequently turned off, js file was not automatically created/updated whenever a display template was created/edited. We wanted to have a search content web part with three lines instead of two lines. We created a new display template by following the steps below. Navigate to Site Settings>Master Pages>Display Templates>Content Web Parts. Download the file Item_TwoLines.html Rename it to Item_ThreeLines.
