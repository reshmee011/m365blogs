The Switch Block - Break, Default, Fail-Through Allows Hierarchical code execution

The Switch Block allows to execute some code based on the value itself without a boolean check. Consider the block of code below function colour (id) { switch(id){ case 1: this.colour = "red"; case 2: this.colour = "blue"; case 3: this.colour = "yellow"; case 4: this.colour = "green"; } } var col = new colour(2); console.log(col.colour); The result of the above code is “green”. The case keyword and its value is just a label for an executable “starting point”.
